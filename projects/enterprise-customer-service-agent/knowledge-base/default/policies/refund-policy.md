# 课程退款政策

## 政策口径

开课一周内可无理由退款；超过一周后，需按已完成课时比例扣除费用。

## 客服处理边界

客服 Agent 只能解释退款政策并建议进入人工审批流程，不能直接承诺退款成功，也不能执行真实退款、取消订单或支付系统操作。

涉及退款、取消、改签等高风险动作时，系统必须先调用只读政策检查工具或查询订单证据，再返回人工审批建议。
