# 企业级 AI Agent 实战课程

## 产品定位

企业级 AI Agent 实战课程面向希望落地智能客服、订单协同、知识库问答和多 Agent 协作的开发者团队。课程基于真实企业项目重构案例，覆盖金融、电商、客服等行业中的 Agent 落地问题。

## 核心能力

- 智能客服对话与意图识别
- 订单查询和只读业务工具调用
- FAQ、政策、产品知识 RAG
- 多 Agent 协作与可视化追踪
- Prompt 约束、安全审批和运行态观测

## 学习结果

完成课程后，学员应具备独立设计和实现企业级 AI Agent 系统的能力，能够交付可运行、可调试、可观测的智能平台原型。
