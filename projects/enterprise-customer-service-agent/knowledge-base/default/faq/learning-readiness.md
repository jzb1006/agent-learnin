# 课程适合哪些学员

## 问题

这门 AI Agent 课程适合零基础学员吗？课程是否偏理论？

## 答案

课程面向有编程经验的开发者、架构师和技术管理者，不建议完全零基础学员直接参加。课程强调企业级落地，会围绕智能客服、订单查询、RAG、多 Agent 协作和可视化追踪等场景提供可运行代码和调试任务。

对于已有项目经验但 AI Agent 经验较少的学员，可以先完成前置自学资料，再进入正式课程。课程目标是让学员结业后能独立搭建一个支持 RAG、多 Agent 协作和可视化追踪的智能平台原型。
